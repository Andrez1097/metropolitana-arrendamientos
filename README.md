# Metropolitana de Arrendamientos (MVP)

Web + Backend para una inmobiliaria en Medellín.
- Listado de inmuebles con filtros
- Detalle con galería
- Mapa por **zona** (sin dirección exacta)
- Contacto por WhatsApp
- Chatbot básico (reglas) listo para evolucionar

## Requisitos
- Python 3.10+
- VS Code recomendado
